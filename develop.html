如何參與或協助程式開發

已經搬到 Github 上了

專案網頁：http://of.openfoundry.org/projects/744/
PCMan 專案網頁，位於經濟部工業局贊助的 OpenFoundry 開發網站上，包含大部分程式開發相關資訊，包括開發人員聯絡資訊、各種專案相關資訊、以及一個 Issue tracker，用來回報錯誤、追蹤 Bugs 的狀態和紀錄各種工作事項。

任何有興趣參與程式開發的人，歡迎到專案網頁上，透過網頁上的資訊聯絡專案管理者。

程式碼取得方式

本專案程式碼全部使用 Subversion (SVN) 版本控制系統管理，最新的程式碼可以由 http://svn.openfoundry.org/pcman/ 取得。
在 Windows 下存取 Subversion，我們推薦使用 TortoiseSVN 這套軟體，相當方便。

編譯需求

開發環境： Microsoft ® Visual C++ 6.0 with MFC 4.2
(使用 Visual C++ 2003 .NET with MFC 7.0 也能編譯，但生成的執行檔體積會大不少)
(目前不支援 Visual Studio 2005 & 2008，若有需要者須手動修正專案檔)
